# Examples

Use these as style targets, not as fixed templates. Keep the user's real concern, but reroute it into period-safe language.

## Example 1

Raw user question:

```text
为什么一个国家光靠法律不够，还要讲礼乐和教化？
```

Safe rewrite:

```text
【可直说版】
后学窃有所惑：治邦国者，明法禁、设刑罚，固所以止奸弭乱；然先王又必重礼乐教化，不专任刑名。后学未达其故，敢请问：礼乐之设，究竟所以正人心欤，成风俗欤，抑所以辅法之所不逮欤？

【更雅一点版】
后学尝闻，法禁可以止其末，礼乐或足以防其原。然其间本末轻重，末学实未之审。敢请问：先王所以并礼乐与刑政而用之者，其深意安在？

【可续接的话】
1. 后学非谓法不足用，特疑其所及者有分耳。
2. 若专任刑罚，能整其行；然风俗人心，是否尚须别有陶铸之道？
3. 此中先后本末，愿闻先生为后学析之。

【避雷提醒】
- 别说`国家`、`制度设计`、`社会控制`
- 别把话说成后世法家通论
- 稳妥说法用`邦国之治`、`礼乐教化`、`刑政并用`
```

## Example 2

Raw user question:

```text
为什么好的制度执行起来总会变形？
```

Safe rewrite:

```text
【可直说版】
后学尝见：朝廷立法制、定名分，其初意未必不善；然及其施行于下，往往与本旨相左。后学未审其故，敢请问：此失在法意未密，在用人不当，抑在上下情伪相蔽而然？

【更雅一点版】
窃谓立制之初，或本欲求治；及其行之既久，反多失其本意。后学愚惑，愿闻其故：其病在成法乎，在任人乎，抑在风俗人情乎？

【可续接的话】
1. 后学所疑者，不在立法之名，正在行法之实。
2. 若法不离人，则此中关键是否终在得人？
3. 愿闻先生就本末先后，为后学指其归宿。

【避雷提醒】
- 别说`执行力`、`组织摩擦`、`系统失真`
- 不要把问题说成现代官僚技术失灵
- 可以守在`法意`、`任人`、`上下情伪`这一路
```

## Example 3

Raw user question:

```text
如果物质条件不够，光讲仁义有用吗？
```

Safe rewrite:

```text
【可直说版】
后学窃有所疑：岁或不登，仓廪不足，民力亦困；当此之时，徒言仁义，果足以安众心乎？抑必先宽其征敛、纾其困急，而后德教可行？后学愿闻其先后。

【更雅一点版】
若民食未给，而遽责以高义，后学恐其难行。然若尽委之财用，而德教不立，亦未必可久。此中缓急先后，愿闻其详。

【可续接的话】
1. 后学非轻仁义，特疑其施之有时耳。
2. 若衣食未充，则教化所凭者是否亦有所待？
3. 此中所谓先富后教，是否可作一端观之？

【避雷提醒】
- 别说`生产力`、`物质基础`、`经济发展`
- 小心`先富后教`这类后世成句，必要时改成`衣食既足而后教可行`
- 稳妥路线是`仓廪`、`征敛`、`民力`、`德教先后`
```
