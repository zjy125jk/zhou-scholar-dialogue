# Term Map

Use this file when the user's raw wording carries modern abstractions. Translate by semantic lane, not by one-to-one word replacement.

## Core rule

Prefer pre-Qin categories that the elder could discuss naturally:

- `先王之道`
- `礼乐刑政`
- `邦国之治`
- `名分伦序`
- `本末先后`
- `心志好恶`
- `财用生民`
- `治乱兴亡`

## Common mappings

| Modern wording | Safer Zhou or pre-Qin rendering | Notes |
| --- | --- | --- |
| 政治 | `为政之道` / `邦国之治` / `治民之术` | Use moral and administrative framing. |
| 经济 | `生民之计` / `财用之道` / `货殖有无` | Prefer subsistence, resources, and use. |
| 社会秩序 | `上下之分` / `伦序` / `邦家之纪` | Keep hierarchy and order concrete. |
| 法律 | `法禁` / `刑罚` / `赏罚之柄` | Use according to scope; do not overstate codification. |
| 制度设计 | `礼制` / `官制` / `成法` / `旧章` | Convert "design" into established forms. |
| 哲学 | `义理之辨` / `名实之论` / `性情之说` | Avoid the modern disciplinary label. |
| 逻辑 | `推其理` / `辨其说` / `考其所由` | Prefer reasoning verbs over nouns. |
| 心理 | `心志` / `好恶` / `情伪` / `志意` | Keep it moral-mental, not clinical. |
| 方法论 | `取法` / `为学次第` / `所守之法` | Make it about approach or rule. |
| 结构 | `条贯` / `本末` / `次第` | Use relation and ordering. |
| 理论 | `所持之说` / `其说` / `大旨` | Keep it light. |
| 效率 | `迟速` / `事功` / `省费而有成` | Use outcome and effort. |
| 信息 / 数据 | `见闻` / `事验` / `典册所载` | Tie to observation or records. |
| 意识形态 | `所宗之说` / `所尚之义` | Make it a school or governing preference. |
| 个人主义 | `各任私意` / `自私自用` | Use only when the user really means private preference. |
| 国家能力 | `朝廷号令之行` / `上下使令之达` | Avoid abstract state-capacity language. |
| 文化传播 | `教化流行` / `诗书之传` / `风俗渐被` | Use transmission through teaching and custom. |
| 历史规律 | `兴亡之故` / `治乱之几` / `成败之由` | Prefer causes and tendencies. |

## Unsafe habits

- Do not transliterate a modern discipline into classical-looking words and assume it is safe.
- Do not keep abstract nouns when a concrete relation or process is available.
- Do not choose a sharp Warring States debate register unless the user wants it.
